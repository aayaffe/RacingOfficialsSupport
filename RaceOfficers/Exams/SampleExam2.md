*Based on the Racing Rules of Sailing (RRS) 2025–2028*

## Quiz #1: Regatta Organization & RC Responsibilities

**1. In the USA, who can be the organizing authority for an event?**
- [ ] Only World Sailing
- [x] **Any organization affiliated with US Sailing**
- [ ] Only a yacht club
> **Reference:** RRS 89.1

**2. The organizing authority for the event appoints the race committee.**
- [x] **True, unless one is appointed by World Sailing**
- [ ] True, unless one is appointed by the technical committee
- [ ] Always true
- [ ] False
> **Reference:** RRS 89.2(c)

**3. Who is responsible for publishing the Notice of Race (NOR)?**
- [ ] The host club
- [x] **The Organizing Authority**
- [ ] The Race Committee
> **Reference:** RRS 89.2(a)

**4. Who can appoint a protest committee?**
- [ ] The race committee
- [ ] The organizing authority
- [ ] National Authority
- [x] **All of these**
- [ ] None of these
> **Reference:** RRS 91

**5. Which of the following can NOT appoint an International Jury?**
- [ ] The organizing authority
- [x] **The race committee**
- [ ] World Sailing
> **Reference:** RRS 91(b)

**6. The scorer is part of ______________.**
- [ ] the series
- [x] **the race committee**
- [ ] the technical committee
- [ ] the protest committee
> **Reference:** Terminology (RRS Introduction): Race Committee

**7. The Sailing Instructions must ______________________.**
- [x] **be made available to each boat before a race begins**
- [ ] be posted online at least 30 days before the event begins
- [ ] be printed and distributed at the competitors' meeting
- [ ] be made available to each boat before a race begins, and readily available throughout the event.
> **Reference:** RRS 25.1

**8. In order to make a valid change to a racing rule, the notice of race or sailing instructions ______________.**
- [ ] must be amended 30 days before the event begins
- [ ] must be approved by US Sailing
- [x] **must refer specifically to the racing rule and state the change**
> **Reference:** RRS 85.1, 86.1

**9. Who is responsible for publishing the Sailing Instructions?**
- [x] **The race committee**
- [ ] The host club
- [ ] The class
- [ ] The organizing authority
> **Reference:** RRS 90.2(a)

**10. Sailing instructions must conform to _________.**
- [ ] Appendix A
- [x] **Appendix J 2**
- [ ] Rule 85.1
- [ ] Appendix J 1
> **Reference:** RRS 90.2(a)

**11. The sailing instructions must contain which of the following? (Select ALL that apply)**
- [ ] a description of the event T-shirt
- [x] **a description of the marks**
- [x] **a description of the class flags**
- [x] **a description of the starting and finishing lines**
> **Reference:** Appendix J2

**12. Changes to the sailing instructions can be made by announcing them at the competitors’ meeting ashore, if so authorized by the sailing instructions.**
- [ ] Always True
- [ ] True, as long as authorized by the NOR
- [ ] True, as long as authorized by the SIs
- [x] **False**
> **Reference:** RRS 90.2(c). *Changes ashore must be in writing. Oral changes are only allowed on the water.*

**13. Class rules are in effect, even if the sailing instructions don’t mention them.**
- [x] **True**
- [ ] False
> **Reference:** Definition: *Rule* (d)

## Quiz #2: The Start Sequence

**1. Starting lines should be between 1 and 1.5 times the aggregate length of the boats starting.**
- [ ] True
- [x] **False**
> **Reference:** Race Management Manual L3

**2. Higher performance boats need _______ starting lines.**
- [ ] shorter
- [x] **longer**
- [ ] reaching
> **Reference:** Race Management Manual L3

**3. In light air, the starting line should be ________.**
- [ ] unfair
- [ ] longer
- [x] **shorter**
> **Reference:** Race Management Manual L3

**4. If the RC wants boats to come within hail, for example to check them in, they should display ______.**
- [x] **the L flag**
- [ ] the AP flag
- [ ] the AP flag over A
- [ ] code flag X
> **Reference:** Race Signals

**5. Unless the Notice of Race or Sailing Instructions change the interval, the warning signal is made ____ minutes before the start.**
- [ ] 1
- [ ] 4
- [x] **5**
- [ ] 10
> **Reference:** RRS 26

**6. The visual signals for starting must be flags, unless changed by the sailing instructions.**
- [ ] True
- [x] **False**
> **Reference:** RRS 25.3 (*Can be flags or "other object of a similar appearance"*)

**7. The class flag (warning signal) must be ____________.**
- [ ] the official class insignia
- [ ] a numeral pennant
- [ ] the same color as the marks of the course
- [x] **described in the Sailing Instructions**
> **Reference:** RRS 26

**8. No later than the ________ signal, the race committee shall signal or otherwise designate the __________ to be sailed.**
- [ ] starting; race
- [ ] preparatory; fleet
- [ ] preparatory; course
- [x] **warning; course**
> **Reference:** RRS 27.1

**9. When the race committee designates the course to be sailed, it is required to display the direction to the first mark.**
- [ ] True
- [x] **False**
> **Reference:** RRS 27.1

**10. The race committee is permitted to move a starting mark only until the __________.**
- [ ] warning signal
- [ ] end of the race
- [x] **preparatory signal**
- [ ] penalty flag is in effect
> **Reference:** RRS 27.2

**11. The preparatory signal is made _____ minute(s) after the warning signal.**
- [x] **1**
- [ ] 4
- [ ] 5
- [ ] 10
> **Reference:** RRS 26

**12. Unless a starting penalty is to be in effect, the preparatory visual signal is**
- [x] **the P-Flag.**
- [ ] a plain blue flag.
- [ ] the I-Flag.
- [ ] the class flag.
> **Reference:** RRS 26, Race Signals

**13. The preparatory signal is lowered 1 minute before the start, with one long sound signal.**
- [x] **True**
- [ ] False
> **Reference:** RRS 26

**14. The starting signal is the _______________.**
- [ ] removal of the preparatory signal
- [ ] removal of the orange line flag
- [ ] removal of the penalty flag
- [x] **removal of the warning signal**
> **Reference:** RRS 26

**15. If more than one boat is identified as OCS, the race committee should ________.**
- [x] **promptly display the X-ray flag with 1 sound**
- [ ] signal a general recall
- [ ] score them both as DSQ
- [ ] "dip" the X-ray flag as each returns to the pre-course side of the line
> **Reference:** RRS 29.1

**16. When flag X has been displayed immediately after a starting signal, _______________.**
- [ ] it must always remain displayed for four minutes
- [ ] it must be displayed until all signaled boats have restarted correctly
- [x] **it must be displayed until all signaled boats have restarted correctly or for four minutes, whichever is sooner**
- [ ] it must be displayed until all OCS boats have finished
> **Reference:** RRS 29.1

**17. The race committee is permitted to signal a general recall if there has been an error in the starting procedure.**
- [x] **True**
- [ ] False
> **Reference:** RRS 29.2

**18. If there are several OCS boats, but only 3 are identified, the race committee should ____________.**
- [ ] give up and have lunch
- [x] **signal a general recall**
- [ ] signal an individual recall and rely on the sailors' guilty consciences
- [ ] shorten the race so that they can run more races
> **Reference:** RRS 29.2

**19. During a fleet race, one minute after lowering the general recall flag (first substitute), the race committee should _____________.**
- [ ] make the preparatory signal
- [ ] raise the X-ray flag
- [x] **signal the warning, AP, or N**
- [ ] make the attention signal
> **Reference:** RRS 29.2, 27.3

**20. When the I-flag penalty is in effect and a boat is on the course side of the start line or its extensions at the start, ______________.**
- [x] **flag X should be displayed immediately after the start, with 1 sound**
- [ ] the race committee shall signal a general recall
- [ ] the black flag should be displayed immediately, with 1 sound
- [ ] she is not breaking a rule and should continue racing
> **Reference:** RRS 29.1

**21. Z-flag penalties apply only if there is a general recall.**
- [ ] True
- [x] **False**
> **Reference:** RRS 30.2 (*The penalty applies if the race continues OR if there is a restart*)

**22. During a Z-flag start that is neither postponed nor abandoned prior to the start, which of the following is not true?**
- [ ] A boat that is OCS at the start must return to the pre-course side of the starting line to properly start. (*True - she must start to sail the race*)
- [x] **A boat that is OCS at the start shall be scored ZFP and receive 1 point more than the number of boats entered in the series.** (*False*)
- [ ] A boat that is over the line at ten seconds must return to the pre-course side of the starting line to properly start. (*True*)
- [ ] If a boat is OCS at the start, the Race Committee shall display flag X with one sound. (*True*)
- [ ] A boat may receive multiple 20% scoring penalties. (*True - in subsequent restarts*)
> **Reference:** RRS 30.2

## Quiz #3: During & Post Race

**1. After the starting signal, the RC may abandon a race because______.**
- [ ] of reasons directly affecting the safety or fairness of the competition
- [ ] light wind makes it unlikely that any boat will finish within the time limit
- [ ] a mark is missing or out of position
- [ ] of foul weather
- [x] **All of these**
> **Reference:** RRS 32.1

**2. As soon as one boat has rounded the first mark, the R/C can no longer abandon the race.**
- [ ] True
- [x] **False**
> **Reference:** RRS 32.1

**3. The R/C is permitted to shorten the course so that further scheduled races can be sailed.**
- [x] **True, as long as no other rules prohibit it.**
- [ ] False
> **Reference:** RRS 32.1

**4. In order to shorten a race by eliminating one or more legs, the race committee is required to make the______.**
- [ ] announcement over VHF
- [ ] appropriate signals at the mark that begins the leg to the new finish mark
- [ ] announcement at the competitors/ meeting
- [x] **appropriate signals at the new finish mark**
> **Reference:** RRS 32.2

**5. When signaling a change in the next leg of the course, the RC displays ___ with ___ as the first boat approaches.**
- [ ] no flags; no sounds
- [ ] S flag; 2 sounds
- [ ] class flag; 3 sounds
- [x] **C-flag; repeated sound signals**
> **Reference:** RRS 33

**6. When a mark is missing, the race committee is permitted to _______.**
- [ ] substitute it with a new one of similar appearance
- [ ] substitute it with a boat/object displaying flag M and make repetitive sounds
- [ ] replace it in its correct position
- [x] **All of these**
> **Reference:** RRS 34

**7. If no time limit is stated in the sailing instructions, then _______.**
- [ ] boats must finish by sunset
- [ ] the race must be abandoned
- [x] **there is no time limit**
- [ ] boats not finished before the party starts shall be scored DSQ
> **Reference:** RRS 35

**8. If no boat finishes within the time limit, the race committee _______.**
- [ ] may abandon the race
- [x] **is required to abandon the race**
- [ ] is fired
- [ ] shall finish the boats in their mark-rounding order
> **Reference:** RRS 35

**9. When a course is shortened, the finish line is _____.**
- [ ] at a line the course requires boats to cross
- [ ] between the staff displaying flag S and the nearby rounding mark
- [ ] at a gate, between the gate marks
- [x] Any of these
> **Reference:** RRS 32.2

**10. Boats are required to completely cross the line after finishing**
- [ ] True
- [x] **False**
> **Reference:** Definition: Finish

**11. If any boat has properly started, sailed the course, and finished within the time limit, the race committee is required to score all boats, including those that finish after the time limit.**
- [x] **True, unless the race is abandoned**
- [ ] True, unless the SIs include a finish window
- [ ] Always true
- [ ] Always false
> **Reference:** RRS 35

**12. The race committee is permitted to protest a boat if a member of the race committee saw the boat hit a mark.**
- [x] **True**
- [ ] False
> **Reference:** RRS 60.2(a)

**13. If the race committee intends to protest a boat for hitting a mark, it must ___________.**
- [ ] fly a red flag
- [ ] chase her down until she completes a tack and a gybe
- [ ] hail that boat immediately
- [x] **notify her within the protest time limit**
> **Reference:** RRS 60.2

**14. If the race committee sees a boat pass a rounding mark on the wrong side, it can score her as NSC.**
- [x] **True**
- [ ] False - it must score her as DNF
- [ ] False - it must score her as DSQ
- [ ] False - it must protest her
> **Reference:** RRS A5.1 (*NSC = Did Not Sail Course. A5.1 allows this without a hearing.*)

**15. A boat that fails to finish in accordance with the definition ________ .**
- [ ] must be protested by another competitor
- [x] **can be scored DNF without a hearing**
- [ ] must be protested by the race committee
- [ ] must be protested by the protest committee
> **Reference:** RRS A5.1

**16. When a boat is disqualified after finishing, __________________ .**
- [x] **each boat with a worse finishing place must be moved up one place**
- [ ] she may not race the next race.
- [ ] each boat with a worse finishing place is given 1 extra point
- [ ] she cannot exclude this score
> **Reference:** RRS A6.1

**17. Unless changed in the sailing instructions, a boat's worst race score, except a/an _____, is excluded from her series score.**
- [ ] NSC
- [x] **DNE**
- [ ] RDG
- [ ] DSQ
> **Reference:** A2.1

**18. When the Race Committee discovers it has scored a boat incorrectly, they must ______________.**
- [x] **correct the error**
- [ ] abandon all races
- [ ] request redress for all boats affected
- [ ] ask the protest committee for their advice
> **Reference:** RRS 90.3(c)
